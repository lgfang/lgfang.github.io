3 Fly check

3.1 flycheck

Prefer "flycheck" to "flymake". The former is very easy to set up and use.

  • Tips for fly-checking shell scripts
    • Tell Emacs where shellcheck is. 
      (setq-default flycheck-sh-shellcheck-executable
              "/db/home/jenkins/exttools/bin/shellcheck")

      shellcheck is a very nice tool for this task. If you installed it in a non-standard place (not in PATH), you can let emacs know using aforementioned configure.

    • Make shellcheck works for ksh 
      (when (eq system-type 'gnu/linux)
  (setq sh-alias-alist
        '((csh . tcsh)
          (ksh . ksh88)
          (bash2 . bash)
          (sh5 . sh))))

      This revised version of sh-alias-alist maps "ksh" (#!/bin/ksh or alike) to "ksh88" instead of "pdksh", which is the default one on linux. The default behaviour creates a mismatch bewteen Emacs shell script mode and flycheck mode since flycheck only accepts "bash", "ksh88", "sh", and "zsh" (In fact, the underlining shellcheck accepts all of "ksh", "ksh88", "ksh93". ShellCheck/Analytics.hs).
